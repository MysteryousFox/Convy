# Convy

A universal file converter that runs entirely in your browser. No servers, no uploads, no privacy concerns — everything is processed locally on your device.

Made by @MysteryousFox
---

## Features

### Images
Convert between PNG, JPG, WebP, GIF, BMP, AVIF, TIFF, SVG and ICO (16×16, 32×32, 64×64). Adjustable quality for JPG and WebP, batch processing, and compression statistics.

### Audio
Convert audio files to MP3, WAV, OGG and WebM powered by ffmpeg.wasm. Three quality presets. Converts instantly in the background — no playback required.

### Video
Convert video files to MP4, WebM, AVI, MOV and GIF, also powered by ffmpeg.wasm. Adjustable quality via CRF. Supports MP4, WebM, MOV, AVI and MKV as input.

### Models
Convert 3D models between OBJ, STL, PLY, GLTF, GLB and FBX (ASCII). Geometry is parsed and exported directly in the browser using Three.js.

### Colors
Live color converter between HEX, RGB, HSL, HSV and CMYK. All formats update in real time as you type. Includes a color picker and a one-click HEX copy button.

### QR Code
Generate QR codes from any text or URL. Customize size, error correction level, foreground and background color. Download as PNG or copy to clipboard.

### Encoding
Text encoding and transformation toolkit: Base64, URL encoding, HTML escaping, Hex, Binary, ROT13, SHA-1 / SHA-256 / SHA-512 hashing, Morse code, and more.
